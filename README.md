## 1.0 markdown(md)란,
- 보통 `README.md`와 같이 서식이 있는 텍스트 파일을 말한다
- 간단한 문법구조로 빠르게 컨텐츠를 작성할 수 있다
- md파일을 HTML로 구현하기 용이하다

## 2.0 헤더(Header)
- `#`으로 시작하며 제목을 강조하는데 쓰인다
- `#`은 HTML 태그의 `<h1>`과 대응하며 `#`의 개수가 늘어날수록 제목의 크기가 작아진다
  - `##` = `<h2>`
- 예시) `# 제목`
